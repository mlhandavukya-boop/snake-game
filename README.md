# 🐍 Snake Game

A classic Snake game built using Python and the Turtle graphics library.
This project demonstrates object-oriented programming, event handling, and game loop logic in a simple and interactive way.

---

 Features

* Smooth snake movement using arrow keys
* Dynamic food generation
* Score tracking system
* Game over detection (wall and self-collision)
* Clean modular code structure

---

Technologies Used

* Python 3
* Turtle (built-in Python graphics library)

---

 How to Run the Game

 1. Clone the repository

```bash
git clone https://github.com/mlhandavukya-boop/snake-game.git
cd snake-game
```

2. Run the game

```bash
python main.py
```

---

Controls

* ⬆️ Up Arrow → Move Up
* ⬇️ Down Arrow → Move Down
* ⬅️ Left Arrow → Move Left
* ➡️ Right Arrow → Move Right

---

Project Structure

```
snake-game/
│── main.py          # Main game loop
│── snake.py         # Snake behavior and movement
│── food.py          # Food generation logic
│── scoreboard.py    # Score tracking system
│── data.txt         # High score storage (if used)
│── .gitignore       # Ignored files
```
